Temas de clase


  • Patrón Arquitectural MVC

    Este tema abarca lo siguiente:
    -Generalidades de Estilos y patrones arquitectónicos.
    -Patrón arquitectónico Modelo-Vista-Controlador servidor de Aplicaciones(Glassfish).
    -Repositorios (Git haciendo uso de Github).

  • Modelo utilizando JPA

    Este tema abarca lo siguiente:
    -Configuración con bases de datos relacionales (Derby, MySQL).
    -Java Persistence API–JPA.
    -Anotaciones Operaciones Transaccionales.

  • Controladores: REST, EJB, PATRONES (DAO/DTO)

    Este tema abarca lo siguiente:
    -Construcción de la lógica de negocio.
    -Patrón de transferencia de información – DTO(Data Transfer Object)
    -Construcción de capa de interconexión mediante la construcción de servicios REST.
    -Manejo de postman.

  • Vista: HTML/CSS/JS (Jquery, Angular)

    Este tema abarca lo siguiente:
    - Tecnología FrontEnd: HTML, CSS(bootstrap, manejo deplantillas debootswatch.)
    -Frameworks de Javascript Jquery.
    -Frameworks de Javascript AngularJS.
